shapefile is generic name for format
binary (opaque)
various files with extensions:
.shp feature geometry
.shx positional index
.dbf columns

various other files
.prf etc

all with same fn, in directory with that fn
usually comes zipped

x.shp.zip 

alternative formats, e.g.

GeoJSON  like json but for geo
